1. Baixar o código do Github ou pelo Moodle:
"git clone https://github.com/josenorbs22/compiladores.git"

2. Instalar a biblioteca PLY:
"pip install ply"

3. Modificar ou criar o arquivo de entrada "input.txt" com o código a ser compilado.

4. Usar o executável do Python ou o comando "python3" com o endereço absoluto ou relativo do arquivo "compilador.py":
"python3 compilador.py"
